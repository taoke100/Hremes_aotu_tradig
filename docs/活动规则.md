参赛须知

本活动不向特定地区用户开放。登录后若页面显示“您所在的地区暂不支持该活动”，则表示当前账户不在本次活动开放范围内，无参赛资格。

1. 比赛时间： 2026/04/09 17:00:00 ~ 2026/04/23 17:00:00。

2. 竞赛形式：个人赛。须通过欧易官方 Agent Trade Kit，将 Claude、GPT 等 AI 模型接入参赛账户，由 AI 在账户中自动执行 USDT 永续合约交易。

3. 计入统计的交易：仅限经由 Agent Trade Kit 自动执行的 USDT 永续合约 (开仓与平仓均计入交易额统计)。以下情况不计入：在欧易 App 或网页端手动下单；使用第三方工具或直接调用 API 下单；现货买卖、期权及币本位合约等其他品种。

4. 上榜门槛 (两项须同时满足)：① 参赛账户 (交易账户) 总资产达到 ≥ 500 USDT ② 达标后，交易的 USDT 永续合约累计成交额 ≥ 1,000 USDT 报名成功后，参赛账户-交易账户总资产首次达到 500 USDT 时视为激活，系统从该时刻起开始统计。激活前的交易不计入排名。

5. 收益额：大赛期间经由 Agent Trade Kit 执行的 USDT 永续合约净盈亏 (不含资金费)，含当前持仓的浮动盈亏 (以实时标记价格计)。

6. 收益率 = AI 交易净盈亏 (含浮动盈亏) / 实际最大净投入 × 100%
每次向交易账户划入资金均会增大分母，从而拉低收益率。若划出后再划入，已划出的资金不重复计入，只算净超出划出金额的新增部分。
示例：划入 2,000 → 划出 2,000 → 再划入 500，分母仅增加 500，而非 2,500。
ⓘ 此处“划入/划出”以参赛交易账户的实际资金变动为准，包括自身资金账户划转、主账户向子账户拨款等所有转入转出。链上充币默认到账资金账户，不直接进入交易账户，不影响收益率计算。

7. Skill 提交截止：2026/04/30 17:00:00。可在赛事期间或结束后提交，不必提前提交。允许重复提交 Skill，若重复提交，则以最后一次提交的为准。未在截止日期 2026/04/30 17:00:00 前提交者无法参与任何奖项评审 (含收益率榜、收益额榜、Skill 精选奖)。

8. 双榜奖励规则：若同时进入收益率榜和收益额榜获奖名次，仅发放金额较高的一项，另一榜该名次空出不补。Skill 精选奖可与排行榜奖励叠加。

9. 账户绑定：报名时选定的账户 (主账户或子账户) 不可更换。同一身份下的多个账户视为同一参赛者，取跨榜最高排名计算奖励。

10. 奖励发放：所有奖励将在赛事结束后 14 个工作日内发放至您的欧易账户。Mac Mini 及 Token 奖励以等值 USDT 形式发放，折算标准：Mac Mini (16GB) = 600 USDT，1 亿 Token = 200 USDT。上述折算标准以活动公告时为准。若发放时市场价格与上述标准存在重大偏差，欧易有权按届时市价进行调整，并提前公告。

11. 作弊处理：以下行为构成作弊，即时取消资格并收回已获奖励：
① 多账户对敲：用同一身份下的多个账户相互对敲交易，虚增收益额或交易额。
② 刷单对冲：故意用其他账户向参赛账户喂单，造假收益。
③ 伪造 AI 交易：手动下单但技术手段伪装成 AI 交易。
④ 策略不符：提交的 Skill 与实际交易行为存在重大出入 (收益额前 10 名将被人工复审)。
⑤ 账户转借：借用他人账户参赛。

12. 排名调整：赛中或赛后，若参赛者被取消资格或账户被封控，其名次自动空出，下方有效参赛者依次上移。赛后资格核查完成后，系统根据有效排行榜自动生成最终获奖名单，补位者获得对应名次的奖励。
