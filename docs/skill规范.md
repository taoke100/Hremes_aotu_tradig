# 策略名称
多信号融合 AI 选标的策略 V1.0
 
# 执行节奏
每 4 小时触发一次
 
# Step 1 · 行情数据采集
调用 market_get_candles 分别获取 BTC-USDT-SWAP / ETH-USDT-SWAP / SOL-USDT-SWAP 的 4h K线，计算 EMA20、EMA60、RSI (14)、ATR (14)。
 
# Step 2 · 市场情绪采集
调用 market_get_funding_rate 获取三个标的资金费率。
调用 market_get_open_interest 获取持仓量变化。
 
# Step 3 · AI 综合判断 (核心)
基于以上数据，你作为交易 AI 需要推理：
1. 三个标的中，哪个趋势信号最强、最清晰？
2. 目标标的资金费率是否过度拥挤 (绝对值 > 0.1%)？若是，仓位减半。
3. ATR 是否处于合理区间？过高则风险过大，过低则机会不足，均应降低仓位或跳过。
4. 综合以上三点，给出最终决策：
开仓/降仓观望/本次跳过，并说明理由。
 
# Step 4 · 执行下单 (仅当 AI 判断明确开仓时)
调用 swap_place_order：
instId = < AI 在第三步选定的标的 >
side = < AI 判断的方向，buy/sell >
ordType = "market"
sz = < 按第三步建议仓位比例计算 >
tag = "agentTradeKit" - 必填，否则不计入排行榜
 
# Step 5 · 止损设置
开仓后立即调用 swap_place_algo_order 设置止损：
多头止损 = 开仓价 × 0.98
空头止损 = 开仓价 × 1.02
 
# 风控规则
// 单笔最大亏损不超过账户净值 3%
// 当日净值回撤超 8% 则停止新开仓
// 最多同时持有 2 个标的，禁止对冲持仓
