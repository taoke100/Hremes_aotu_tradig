# Hermes AI Trading Kit

TypeScript 版本的加密货币自动交易机器人，支持多交易所、多交易员管理。

## 快速开始

```bash
git clone https://github.com/taoke100/Hremes_aotu_tradig.git
cd Hremes_aotu_tradig
git checkout v1.3.1-ts
npm install
PORT=8889 node --import tsx src/server.ts
```

## API 端口

- TypeScript 版：8889
- Python 版：8888（暂不可用）

## 主要功能

- 多交易员管理（CRUD）
- 实时交易状态监控
- AI 交易引擎集成
- 配置文件自动持久化
